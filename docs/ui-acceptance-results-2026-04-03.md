## 人工验收执行顺序（建议）
1. 平台登录后逐页点击菜单，按 390x844 检查：无横向溢出、按钮可点、弹窗可操作。
2. 同样流程切换到批发商、零售商，确认只显示各自菜单。
3. 每个角色刷新当前页，确认不掉登录且停留在当前页。
4. 最后在 1366x768 复测关键页面：POS、仓库管理、财务看板。
